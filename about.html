The 50 CSS Challenge

Sobre o desafio

The 50 CSS Challenge é um desafio que consiste em fazer 50 mini-projetos com a linguagem CSS. O objetivo do desafio é praticar, aprender e compreender a linguagem CSS.
Os projetos são feitos com a ajuda de professores que ensinam de forma gratuita através de tutoriais em plantaformas como o youtube (90% dos projetos), blogs e instagram.

Regras

  • Prazo máximo de 3 meses a contar do dia de começo (Mais ou menos 1 projeto em cada 2 dias).
  • O codígo é sempre reescrito, não é permitido o "Copy & Paste".
  • Os projetos podem conter Javascript desde que o foco seja a pratica do CSS
